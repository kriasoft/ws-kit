View on GitHub
🛡️

Type-Safe Routing

Define message schemas with Zod or Valibot and get full TypeScript type inference from schema to handler without type assertions

🔄

Request-Response Pattern

Built-in RPC with auto-correlation, timeouts, and streaming progress updates using rpc() helper and ctx.reply()

📡

Broadcasting & PubSub

Type-safe publish/subscribe with topic-based routing. Use ctx.publish() in handlers or router.publish() for system events

🔌

Middleware Support

Global and per-route middleware with async/await support for authentication, rate limiting, and authorization

⚡

Multi-Platform

Works with Bun's high-performance WebSocket server and Cloudflare Durable Objects with platform-specific optimizations

🔧

Structured Error Handling

13 gRPC-aligned error codes with WsKitError class following WHATWG Error standard. Automatic error responses, cause chaining, and JSON serialization for observability tools
