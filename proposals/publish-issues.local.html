On this page

The dependencies in the package.json files of the published packages are using workspace:* instead of the correct version numbers. This will prevent the packages from being used correctly.

  • @ws-kit/valibot: dependencies has "@ws-kit/core": "workspace:*", should be "@ws-kit/core": "^0.3.0".
  • @ws-kit/zod: dependencies has "@ws-kit/core": "workspace:*", should be "@ws-kit/core": "^0.3.0".
  • @ws-kit/redis-pubsub: dependencies has "@ws-kit/core": "workspace:*", should be "@ws-kit/core": "^0.3.0".
  • @ws-kit/client: dependencies has "@ws-kit/valibot": "workspace:" and "@ws-kit/zod": "workspace:", should be "@ws-kit/valibot": "^0.2.1" and "@ws-kit/zod": "^0.2.1".
  • @ws-kit/cloudflare-do: dependencies has "@ws-kit/core": "workspace:*", should be "@ws-kit/core": "^0.3.0".
  • @ws-kit/bun: dependencies has "@ws-kit/core": "workspace:*", should be "@ws-kit/core": "^0.3.0".
  • @ws-kit/core: No issues found.
